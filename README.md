# Around The U.S.

## Descripción del proyecto

Around The U.S. es una aplicación web interactiva desarrollada como parte del Sprint 6 del bootcamp de desarrollo Full Stack de TripleTen.

El proyecto permite a los usuarios interactuar dinámicamente con tarjetas de fotografías mediante JavaScript, incluyendo funciones para:

* Editar información del perfil
* Crear nuevas tarjetas
* Dar “me gusta” a las tarjetas
* Eliminar tarjetas
* Abrir imágenes en tamaño completo dentro de una ventana emergente

La aplicación utiliza manipulación del DOM, plantillas HTML (`<template>`), formularios interactivos y manejo dinámico de eventos.

## Funcionalidades

### Perfil

* Edición dinámica del nombre y descripción del perfil mediante un formulario emergente.

### Tarjetas dinámicas

* Renderizado de tarjetas desde un array de objetos utilizando plantillas HTML.
* Generación dinámica de nuevas tarjetas desde un formulario.

### Interactividad

* Botón de “Me gusta” funcional.
* Eliminación de tarjetas del DOM.
* Vista ampliada de imágenes en un modal.

### Modales reutilizables

* Funciones reutilizables para abrir y cerrar ventanas emergentes.
